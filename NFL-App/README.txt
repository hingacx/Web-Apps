NFL Theme'd Web App with a focus on front end development.

* Looks up an NFL Team's roster and displays the current players.
* Has a player tracker that keeps track of players and their unique player IDs
* Individual Player search functionality if the unique ID is known.


*** INSTRUCTIONS ***
1. Create a new directory
2. Load all of the files except the README.txt into the directory
3. Launch index.html using chrome
4. Enjoy! (Don't abuse the API calls please)
